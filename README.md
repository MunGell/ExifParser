# ExifParser

Simple EXIF data parser.

This library is not yet listed at Packagist, but you can install it using this code in your `composer.json` file:

```
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/MunGell/ExifParser"
        }
    ],
    "require": {
        "MunGell/ExifParser": "*"
    },
}
```
